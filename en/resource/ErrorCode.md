## Error Code

Explanation of error code returned when loading panel fails:

| Code | Description                            |
| ---- | -------------------------------------- |
| 1901 | Unsupported Device                     |
| 1902 | Group is empty                         |
| 1903 | Not set homeId                         |
| 1904 | Device is nil                          |
| 1905 | RN resource not found                  |
| 1906 | Present VC is not NavigationController |
| 1907 | Unknow                                 |
